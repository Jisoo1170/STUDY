# Git 기초 명령어



## 공유

```cmd
git config --global user.name "이름"
git config --global user.email "이메일"
git clone 공유할 주소
```



## 업로드

~~~cmd
git add .(모든거)/파일명
git commit -m "설명"
git push (-u origin master 처음에만)
~~~
